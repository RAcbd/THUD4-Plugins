# Current Plugins Available

## MonsterOnmapDamageType
+ Shows monsters on the map and differentiates between Elite and Regular monsters.
+ Uses MonsterDamageType in colors to differentiate monster attack types
    Cold: Blue
    Fire: Red
    Lightning: Yellow
    Physical: Gray
    Poison: Green
    Shadow: Purple
    None: White
